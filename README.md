## Exercise 3

**Problem:**

Check if number is perfect number.
_Perfect number is a positive integer that is equal to the sum of its positive proper divisors._
_For example 6 is perfect number._

**Example**

 	Input: 6
  	Output: True
 
  	Input: 15
  	Output: False
 
  	Input: 28
  	Output: True
